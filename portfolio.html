Project 1: DevConnect!

DevConnect is a networking app that provides a platform catered for exchanging and sharing ideas among passionate developers. It allows them to connect, share and chat with like-minded enthusiastic developers. The technologies used to develop this networking app are JavaScript, Node.js, Express.js, MongoDB and Bootstrap.

This is a basic games app where you open all of the doors without running into the Lion! A variety of skills were used to make this responsive app including HTML, CSS and JavaScript.

Project 3: Fanur Technology Website!

This is a fully responsive portfolio website. I used HTML, CSS, JavaScript and jQuery to make it. While working on my website, I thought carefully about Keeping my design balanced. I used grids, flex boxes and picked a couple of good base colors to keep my website consistent and professional. I made the graphics go well together and used appropriate typography. I made the elements stand out by adding white space around them. All elements are nicely connected together and fully functional.

Project 4: Todo-App!

Although this is a simple todo-app, it combines the use of front-end and back-end skills to fully connect browser, server and database. I used HTML, CSS and JavaScript for the front-end, Node.js for the server side and MongoDB for the database. This app is fully responsive and uses cross browser compatibility.
